# SG FOREVER - Paquetería Express

Página web moderna y completa para una empresa de paquetería tipo Shalom con el nombre **SG FOREVER**.

## 📋 Contenido

La página incluye todas las funcionalidades principales de un servicio de paquetería:

### ✨ Características

- **Header Sticky**: Navegación siempre visible con logo de SG FOREVER
- **Hero Section**: Sección de bienvenida con llamado a la acción
- **Rastreo de Envíos**: Busca tu paquete por número de guía
- **Cotizador**: Calcula el costo de tu envío en tiempo real
- **Servicios**: Muestra los 6 servicios principales (Express, Internacional, Recolección, Seguros, App Móvil, Planes Corporativos)
- **Sucursales**: Ubicaciones de oficinas en diferentes ciudades
- **Formulario de Contacto**: Para que los clientes se comuniquen
- **Footer**: Información legal y enlaces

### 🎨 Diseño

- **Colores**: Naranja (#ff6b35) y Azul Marino (#004e89)
- **Responsivo**: Se adapta a dispositivos móviles y de escritorio
- **Animaciones**: Efectos suaves al hacer scroll y hover
- **Moderna**: Gradientes, sombras y transiciones elegantes

## 🚀 Cómo Usar

1. Abre el archivo `index.html` en tu navegador
2. Explora las diferentes secciones usando la navegación superior
3. Prueba las funcionalidades:
   - Ingresa un número de guía para rastrear
   - Completa el formulario de cotización
   - Envía un mensaje en la sección de contacto

## 📁 Archivos

- `index.html` - Estructura y contenido HTML
- `styles.css` - Estilos y diseño
- `script.js` - Funcionalidades interactivas

## 🔧 Funcionalidades JavaScript

### Rastreo
- Ingresa un número de guía (máximo 20 caracteres)
- Se muestra la historia del envío con timeline

### Cotizador
- Ingresa origen, destino, peso y tipo de servicio
- Calcula el costo automáticamente
- Tres tipos de servicio: Express, Regular y Económico

### Formulario de Contacto
- Validación de email
- Mensaje de confirmación al enviar

## 📱 Responsivo

La página se adapta automáticamente a:
- Pantallas de escritorio (1200px+)
- Tablets (768px - 1199px)
- Móviles (menos de 768px)
